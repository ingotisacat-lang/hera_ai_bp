## 🚀 如何使用

### 方案 A: 直接演示
1. 打开 `HERA_BP_完整版.html`（中文）或 `HERA_BP_Complete_EN.html`（英文）
2. 按 F11 全屏
3. 使用 ↓/↑ 键或右下角按钮翻页
4. 双击进入/退出全屏

### 方案 B: 导出 PDF
1. 浏览器打开 HTML 文件
2. Cmd+P（Mac）或 Ctrl+P（Windows）
3. 选择"另存为 PDF"
4. 调整边距和缩放

### 方案 C: 转为 PPT
1. 参考 HTML 内容
2. 在 Keynote/PowerPoint 中重制
3. 使用相同配色方案：
   - 背景：`#0a1628` → `#1a365v` 渐变
   - 主色：`#667eea` → `#764ba2` 渐变
   - 辅色：`#00c6ff` → `#0072ff` 渐变
