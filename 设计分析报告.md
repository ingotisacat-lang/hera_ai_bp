## 🎨 原 PPT 设计风格解构

### 1. 色彩系统

**主色调**:
- 背景：深蓝色渐变（#0A1628 → #1A365D）
- 主色：亮蓝色（#00C6FF → #0072FF）
- 辅色：紫色（#667EEA → #764BA2）
- 文字：白色（#FFFFFF）+ 灰色（#A0AEC0）

**渐变应用**:
- 标题文字：蓝紫渐变（#667EEA → #764BA2）
- 卡片背景：半透明渐变（rgba(102, 126, 234, 0.1) → rgba(118, 75, 162, 0.1)）
- 按钮/强调：蓝色渐变（#00C6FF → #0072FF）

**色彩比例**:
- 背景色：60%（深蓝）
- 主色：25%（蓝色/紫色渐变）
- 文字色：10%（白色/灰色）
- 强调色：5%（亮蓝/紫色）

---

### 2. 排版系统

**字体**:
- 英文：SF Pro Display / Helvetica Neue
- 中文：PingFang SC / 思源黑体
- 字重：Light (300), Regular (400), Medium (500), Semibold (600), Bold (700)

**字号规范**:
| 用途 | 字号 | 字重 |
|------|------|------|
| 主标题 | 72-96px | Bold (700) |
| 副标题 | 32-40px | Medium (500) |
| 正文 | 18-24px | Regular (400) |
| 注释 | 14-16px | Light (300) |
| 大数字 | 80-120px | Bold (700) |

**行距**:
- 标题：1.1-1.2
- 正文：1.6-1.8
- 列表：1.5

**字间距**:
- 大标题：-2px to -4px（紧凑）
- 正文：0px（正常）

---

### 3. 布局系统

**网格**:
- 12 列网格系统
- 列间距：24px
- 边距：80px（左右）

**卡片设计**:
- 背景：rgba(255, 255, 255, 0.05)
- 边框：1px solid rgba(255, 255, 255, 0.1)
- 圆角：16px-20px
- 内边距：40px-60px
- 阴影：0 8px 32px rgba(0, 0, 0, 0.3)
- 毛玻璃：backdrop-filter: blur(10px)

**间距规范**:
- 页面边距：80px
- 卡片间距：60px
- 元素间距：24px-40px
- 段落间距：40px-60px

---

### 4. 视觉元素

**数据可视化**:
- 大数字：80-120px，渐变文字
- 标签：18px，灰色（#A0AEC0）
- 图表：简洁线条，无多余装饰
- 颜色：蓝紫渐变为主

**图标**:
- 风格：线性图标
- 粗细：2px 描边
- 尺寸：24px, 32px, 40px
- 颜色：白色或渐变

**分割线**:
- 粗细：1px-2px
- 颜色：rgba(255, 255, 255, 0.1)
- 渐变：transparent → 主色 → transparent

### 6. 动画与过渡

**页面过渡**:
- 效果：淡入淡出
- 时长：0.5s
- 缓动：ease-in-out

**元素动画**:
- 进入：从下向上滑入 + 淡入
- 时长：0.3s-0.5s
- 延迟：依次延迟 0.1s

**强调动画**:
- 数字增长：从 0 到目标值
- 卡片悬停：轻微上浮（translateY(-5px)）
