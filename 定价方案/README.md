## 🚀 使用方法

### 桌面浏览器
1. 用 Chrome/Safari/Edge 打开 `V3.6-HTML 版.html`
2. Cmd + P 打印为 PDF（勾选"背景图形"）

### 手机浏览器
1. 用 Safari/Chrome 打开 `V3.7-Mobile 版.html`
2. 添加到主屏幕（PWA 支持）

### 微信内打开
1. 在微信中打开 HTML 文件链接
2. 表格可横向滑动查看
